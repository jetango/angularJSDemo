#Kir_UI

## environment

- node v5.4.1
- npm 3.3.12
- webpack

## used technologies:  

- Node
- Npm
- Webpack
- Angular2.0
- ES6
- TypeScript


## npm scripts

* `npm start` - Build and start the app in dev mode at http://localhost:8000
* `npm run build` - Run a production build


## How to start

Run: 

```
npm install
npm start

```
